# Projeto de Reconhecimento de Texto (OCR)

## Objetivo
O projeto visa implementar um sistema de reconhecimento de texto a partir de imagens.

## Como rodar o projeto
1. Clone este repositório
2. Instale as dependências (se houver)
3. Execute o script principal

## Inputs
As imagens utilizadas para o reconhecimento de texto estão na pasta `inputs`.

## Outputs
O resultado do reconhecimento de texto se encontra na pasta `output`.
